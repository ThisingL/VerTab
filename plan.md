# VerTab - Chrome 垂直标签页扩展

## 项目概述

一个 Chrome 扩展，通过 Content Script 注入的方式在网页侧边实现垂直标签页列表，提供类似 ARC 浏览器的体验。

---

## 核心需求

| 需求项 | 说明 |
|--------|------|
| 触发方式 | 点击扩展图标切换显示/隐藏 |
| 侧边栏位置 | 用户可选择左侧或右侧 |
| 页面行为 | 侧边栏展开时压缩网页内容（推开），不遮挡 |
| 主题 | 暗色/亮色可切换 |
| 功能级别 | 完整功能 |

## 技术架构

```
VerTab/
├── manifest.json          # Chrome 扩展清单 (Manifest V3)
├── background/
│   └── service-worker.js  # 后台服务，管理标签页状态、响应消息
├── content/
│   ├── content.js         # 注入网页的脚本，创建侧边栏 DOM
│   └── content.css        # 侧边栏样式
├── sidebar/
│   ├── sidebar.html       # 侧边栏 UI（作为 iframe 内容加载）
│   ├── sidebar.js         # 侧边栏逻辑
│   └── sidebar.css        # 侧边栏内部样式
├── assets/
│   └── icons/             # 扩展图标
├── options/
│   ├── options.html       # 设置页面
│   └── options.js
└── plan.md
```

### 关键技术决策

1. **侧边栏用 iframe 隔离**
   - Content Script 注入一个 `<iframe>` 到页面中，iframe 加载扩展内部的 sidebar.html
   - 优势：样式完全隔离，不受宿主页面 CSS 影响，也不影响宿主页面
   - iframe src 使用 `chrome.runtime.getURL()` 加载扩展内部资源

2. **通信机制**
   - `sidebar.js` ↔ `background service-worker`：通过 `chrome.runtime.sendMessage` / `chrome.runtime.onMessage`
   - `content.js` ↔ `background`：同上
   - `background` 监听 `chrome.tabs` 事件，向所有 content script 广播标签页变化

3. **页面压缩实现**
   - Content Script 在 `<html>` 元素上添加 margin/padding（如 `margin-left: 280px`）
   - 侧边栏使用 `position: fixed` 固定在视口边缘
   - 通过 CSS transition 实现平滑展开/收起动画

4. **状态管理**
   - 侧边栏的展开/收起状态存储在 `chrome.storage.local`
   - 用户设置（主题、位置、宽度）存储在 `chrome.storage.sync`（跨设备同步）
   - 标签页数据通过 `chrome.tabs` API 实时获取，不做本地缓存

5. **Manifest V3**
   - 使用 Service Worker 替代 background page
   - 使用 `chrome.action.onClicked` 响应图标点击
   - 权限：`tabs`、`storage`、`activeTab`

## UI 设计参考

### 侧边栏布局（自上而下）

```
┌─────────────────────┐
│  🔍 搜索标签页...    │  ← 搜索框
├─────────────────────┤
│  📌 固定的标签页     │  ← 固定区域（可折叠）
│  ├─ 🌐 GitHub       │
│  └─ 🌐 Gmail        │
├─────────────────────┤
│  所有标签页          │  ← 标签页列表（可滚动）
│  ├─ 🌐 Google    ✕  │
│  ├─ 🌐 YouTube   ✕  │  ← 当前活跃标签页高亮
│  ├─ 🌐 Stack..   ✕  │
│  └─ ...              │
├─────────────────────┤
│  ＋ 新建标签页       │  ← 底部操作区
│  ⚙ 设置   🌓 主题   │
└─────────────────────┘
```

### 视觉风格
- 圆角卡片式标签项
- hover 时背景微变 + 显示关闭按钮
- 当前标签页左侧有彩色指示条
- 字体：系统字体栈（-apple-system, system-ui, ...）
- 侧边栏默认宽度：280px
- 动画：展开/收起使用 300ms ease 过渡

## 兼容性注意事项

- 部分网站（如 chrome://、Chrome Web Store）不允许注入 Content Script，这些页面侧边栏不可用
- 极少数网站的严格 CSP 可能阻止 iframe，需要优雅降级（显示提示）
- `about:blank`、PDF 查看器等特殊页面需要特殊处理

## 安装与使用教程

### 安装步骤

1. **打开 Chrome 扩展管理页面**
   - 在地址栏输入 `chrome://extensions/` 并回车
   - 或者点击 Chrome 右上角菜单 → "扩展程序" → "管理扩展程序"

2. **开启开发者模式**
   - 在扩展管理页面右上角，打开「开发者模式」开关

3. **加载扩展**
   - 点击左上角「加载已解压的扩展程序」按钮
   - 在文件选择器中选择 `VerTab/` 项目文件夹（即包含 `manifest.json` 的目录）
   - 点击「选择」确认

4. **固定扩展图标（推荐）**
   - 点击 Chrome 右上角的拼图图标（扩展程序）
   - 找到 VerTab，点击图钉图标将其固定到工具栏

### 使用方法

#### 基本操作

| 操作 | 方法 |
|------|------|
| 打开/关闭侧边栏 | 点击工具栏的 VerTab 图标 |
| 切换标签页 | 点击侧边栏中的标签项 |
| 关闭标签页 | 悬浮标签项后点击右侧 ✕ 按钮 |
| 新建标签页 | 点击侧边栏底部的「+ 新建标签页」 |
| 搜索标签页 | 在顶部搜索框中输入关键词 |

#### 标签页管理

| 操作 | 方法 |
|------|------|
| 拖拽排序 | 长按标签项拖动到目标位置 |
| 固定标签页 | 右键标签项 → 「固定标签页」 |
| 取消固定 | 右键已固定的标签页 → 「取消固定」 |
| 创建分组 | 右键标签项 → 「添加到分组」→ 选择或新建分组 |
| 恢复已关闭标签 | 点击侧边栏底部的「最近关闭」查看并恢复 |

#### 设置与个性化

| 操作 | 方法 |
|------|------|
| 切换主题 | 点击侧边栏底部的主题图标（🌓） |
| 切换侧边栏位置 | 点击底部⚙设置 → 选择「左侧」或「右侧」 |
| 调整侧边栏宽度 | 拖动侧边栏边缘 |
| 更多设置 | 点击底部⚙设置，或右键扩展图标 → 「选项」 |

### 注意事项

- **部分页面不可用**：Chrome 内部页面（`chrome://`开头）、Chrome 网上应用店页面不支持注入扩展，这些页面无法显示侧边栏
- **首次安装后**：已经打开的标签页需要刷新才能使用侧边栏，新打开的标签页会自动生效
- **更新扩展**：修改代码后，回到 `chrome://extensions/` 页面，点击 VerTab 卡片上的刷新按钮（🔄），然后刷新已打开的网页

### 快捷键配置（可选）

1. 进入 `chrome://extensions/shortcuts`
2. 找到 VerTab
3. 为「切换侧边栏」设置你喜欢的快捷键组合（如 `Ctrl+B` 或 `Cmd+B`）
